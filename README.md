1.This program reads a text file. It cleans the words and counts how often they appear. It prints the ten most frequent words.
2.A text file is read and its words are cleaned. Two wo word pairs are formed in order and their counts are recorded. Finally the five most frequent pairs are printed.
3.The file is read line by line. Each line is normalized and grouped with similar lines. Duplicate groups are counted and shown.
4.This program loads a student CSV file. It filters rows by study time internet access and absences. It saves the filtered students to a new CSV and prints the number saved and the average grade.
5.The program reads a student CSV file. It assigns each student a grade band called low medium or high. It then groups the data by band and prints counts averages and activity and internet rates.
6.Crime data is read from a file. Each area is marked as high crime or low crime based on violence level. Average unemployment is shown for both groups.
7.The page is downloaded and parsed. The title is printed first. Then the first long paragraph from the main content is printed.
8.Headings are taken from the page content. Certain sections are skipped. The remaining headings are saved and printed.
9.The page is read and a table is located. Each row of that table is written into a single column in a CSV file. The total number of rows saved is printed.
10.The file is scanned line by line. Lines containing a chosen word are recorded with their line numbers. The total matches and the first three results are printed.
